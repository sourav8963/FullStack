# 🌤️ SkyMeter — Weather & AQI Meter

SkyMeter is a simple and user-friendly web application that provides real-time **weather updates** and **Air Quality Index (AQI)** data for any location. It helps users stay informed about environmental conditions and plan their day better.

---

## 🚀 Features

* 🌡️ **Live Weather Data**

  * Temperature, humidity, wind speed
  * Feels-like temperature
  * Weather conditions (sunny, cloudy, rain, etc.)

* 🌫️ **Air Quality Index (AQI)**

  * Real-time AQI values
  * Air quality status (Good, Moderate, Poor, etc.)
  * Health recommendations based on AQI

* 📍 **Location-Based Search**

  * Get weather and AQI by city name
  * Detect user location (optional)

* 🎨 **Clean UI**

  * Simple and responsive design
  * Easy to read data visualization

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** OpenWeather API / AQI API
* **Deployment:** Vercel / Netlify

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/skymeter.git
   ```

2. Navigate to the project folder:

   ```bash
   cd skymeter
   ```

3. Open `index.html` in your browser

---

## 🔑 API Configuration

* Get your API key from:

  * OpenWeather
  * AQI provider (like WAQI)

* Replace the API key in your JavaScript file:

  ```javascript
  const API_KEY = "your_api_key_here";
  ```
